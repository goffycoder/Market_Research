# Factor Framework

This document expands the factor taxonomy in the main README.

## Design Rule

Every factor should answer at least one of these questions:

- Does it change liquidity?
- Does it change the discount rate?
- Does it change growth expectations?
- Does it change inflation expectations?
- Does it force balance-sheet adjustment?
- Does it change who the marginal buyer or seller is?

If the answer is "no" to all six, the factor is probably noise.

## What Each Bucket Can Predict Best

### Best For 1-5 Day Horizon

- Policy surprise
- Positioning
- Volatility regime
- Social / narrative acceleration
- Geopolitical events
- Funding stress

### Best For 1-4 Week Horizon

- Bond supply
- Auction absorption
- PMIs
- CPI surprise
- earnings revisions
- cross-border flows

### Best For 1-3 Month Horizon

- Central-bank trend
- credit impulse
- fiscal stance
- macro surprise trend
- commodity trend
- currency trend

## Explainability Standard

Every final signal should be explainable like this:

- Which factors moved
- Which countries drove the move
- Which asset classes confirmed it
- Which horizon the move is relevant for
- Which evidence is hard data vs inferred narrative
